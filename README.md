## 🚀 About The Project

🎯 **StreamFlix** is a simple static web application containerized using Docker.
It demonstrates how to deploy a static website using a lightweight web server inside a container.

---

## 🛠️ Tech Stack

* 🌐 HTML, CSS, JS
* 🐳 Docker
* ⚡ Nginx (for serving static files)

---

```
StreamFlix-Static-Application/
│── assets/

│── README.md
```

---

## 🐳 Docker Setup

### 🔧 Build Image

```bash
docker build -t streamflix .
```

### ▶️ Run Container

```bash
docker run -d -p 8080:80 streamflix
```

### 🌍 Access App

👉 Open in browser:
http://localhost:8080

⚠️ Note: This URL works only on your local machine.
